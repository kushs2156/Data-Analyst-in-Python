- Import the `math` package.
- Use `math.pi` to calculate the circumference of the circle and store it in `C`.
- Use `math.pi` to calculate the area of the circle and store it in `A`.
```Python
# Import the math package
import math

# Calculate C
C = 2 * 0.43 * math.pi

# Calculate A
A = math.pi * 0.43 ** 2

print("Circumference: " + str(C))
print("Area: " + str(A))
```
